# Gatling Sample

#### This is a simple project using [Gatling](https://gatling.io/), [Scala](https://www.scala-lang.org/) and [SBT](https://www.scala-sbt.org/).

Start SBT
---------
```bash
$ sbt
```

Run all simulations
---------
```bash
> Gatling/test
```

Run a single simulation
-----------------------

```bash
> Gatling/testOnly br.com.sample.simulation.BasicSimulation
```
